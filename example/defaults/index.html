drop-ng example

drop-ng provides an Angular wrapper around Drop.js.

Drop.js is a Javascript and CSS library for creating dropdowns and other popups attached to elements on the page.

The following text will appear within the drop:

This example has no settings passed from the controller instead using defaults set within the directive.

These are:

classes = 'drop-theme-arrows-bounce'
constrainToScrollParent = true
constrainToWindow = true
position = 'top center'
openOn = 'click'
